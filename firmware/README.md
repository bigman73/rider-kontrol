# Button Mapping

| Description | Usage Short | Usage Long |
|------------------------------------|--|--|
| Button 1 (+)      |Zoom In||
| Button 2 (-) |Zoom Out|Zoom Out 5x|
| Button 3 (P) |Play/Pause Music||
| Button 4 (C) |Center Map|Key map change|
| Button 5 ⬅️   | Scroll map left||
| Button 6 ➡️   | Scroll map right||
| Button 7 ⬆️   | Scroll map up||
| Button 8 ⬇️   | Scroll map down||
